# Interactive Music Player Web Application

A simple music player built using **HTML, CSS, and JavaScript**.
It allows users to play, pause, and navigate songs through an interactive web interface.

## Features

* Play / Pause songs
* Song list navigation
* Simple music player UI

## How to Run the Project

1. Download the project from GitHub.
2. Extract the ZIP file.
3. Open the project folder.
4. Create a folder named **songs** inside the project directory.
5. Add your `.mp3` audio files inside the **songs** folder.
6. Open **index.html** in your browser.

Example folder structure:

Music-Streaming-Web-Player
│
├── index.html
├── style.css
├── script.js
└── songs/ (add your audio files here)

Note: Audio files are not included in this repository due to file size limitations.
